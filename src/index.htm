Bootstrap

Bootstrap - это открытый и свободный фреймворк для разработки веб-приложений. Он содержит HTML, CSS и JavaScript-компоненты для создания пользовательских интерфейсов веб-сайтов и веб-приложений.

Bootstrap был создан Twitter и первоначально выпущен в 2011 году как Twitter Blueprint. В настоящее время Bootstrap поддерживается сообществом разработчиков, и его последняя версия - Bootstrap 5 - выпущена в мае 2021 года.
